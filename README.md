# Zero Labs

A minimal Zola site for `zerolabs.sh`.

## Local development

Run the site locally with:

```sh
zola serve
```

Build the site with:

```sh
zola build
```

## Content layout

- `content/_index.md` is the homepage.
- `content/projects/_index.md` is the projects page.
- `content/services/_index.md` is the services page.
- `content/blog/` contains the blog landing page and posts.
- `templates/` contains the base, section, page, and homepage templates.
- `static/styles.css` contains the site stylesheet.
